Variables

Variables are used to store data that can be used later in the program.

JavaScript uses the keywords varletconst to declare variables.

Example:

let myVar = 5;

The value of myVar is now 5.

Declaring variables

Variables can be declared using different keywords based on the scope and the mutability required.

  • var - Function-scoped or globally-scoped, can be redeclared.
  • let - Block-scoped, cannot be redeclared.
  • const - Block-scoped, read-only constant.

Example:

const PI = 3.14;

The value of PI is constant and cannot be changed.

Game tetris

This is one of the cool things that you can program using JavaScript:

Tetris is a fast-paced puzzle game where differently shaped blocks (called tetrominoes) fall from the top of the screen. Your goal is to rotate and move these pieces to form complete horizontal lines. When a line is completed, it disappears and you earn points. As the game progresses, the pieces fall faster, making it harder to keep the board clear. The game ends when the stack of pieces reaches the top.

Controls:

  • A/D - Move left/right
  • W - Rotate piece
  • S - Move down faster
  • Tab - Hard drop
  • R - Restart game
